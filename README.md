## **Media query mixins**

- **before()**
- **after()**
- **is()**

Uses:

- **mobile**
- **tablet**
- **laptop**
- **desktop**
- **wide**
- **ultrawide**
- **superultrawide**

`is()` also accepts:

- **small**
- **medium**
- **large**

<br />


```scss
// @use not required

@include before(tablet) {
  .example {
    color: blue;
  }
};
```
